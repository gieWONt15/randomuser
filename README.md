# Random User API Test
This is a simple project created by Paweł Gierlotka, one of his first projects while learning to code. The project uses the randomuser.me API to display a random user's information on a webpage.

## Installation
There is no installation required to run this project. Simply download the code or clone the repository and open the index.html file in a web browser.

## Usage
The webpage will display a random user's photo, full name, username, email address, and country of origin. This information is retrieved using the randomuser.me API and displayed on the webpage using JavaScript.
